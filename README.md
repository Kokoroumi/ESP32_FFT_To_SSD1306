# 修改

1、引入头文件 `` #include  <ArduinoFFT.h>``

2、创建FFT对象 ``ArduinoFFT<float> myFFT(vReal, vImag, samples,SAMPLING_FREQUENCY);``

3、adafruit创建的显示对象似乎默认使用硬件 **I2C**，接口请用 **SDA = 21，SCL = 22**
